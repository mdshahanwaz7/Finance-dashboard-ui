# FINANCE DASHBOARD UI
A modern Finance Dashboard built with React that helps users track income, expenses, and gain insights through dynamic charts and analytics.

# FEATURES

# Dynamics Charts
 ->Dynamic Charts
->Monthly Income vs Expense (Line Chart)
->Category-wise Spending (Pie Chart)
# Transaction Management
->Add, Edit, Delete transactions
->Filter and search transactions
Sort by amount
# Insights
->Highest spending category
->Monthly comparison
->Smart observations from data
# Summary Cards
->Total Balance
->Total Income
->Total Expenses
# Responsive Design
->Works on mobile, tablet, and desktop
->Sidebar with toggle menu

# TECH STACK
->Frontend: React, Tailwind CSS
->Charts: Recharts
->Routing: React Router DOM
->State Management: useState (React Hooks)
